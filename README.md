# 📊 Data Visualization & Analysis Toolkit

## 🚀 Overview

This project is a comprehensive implementation of data visualization techniques using Python. It covers fundamental to advanced visualizations across multiple domains.

---

## 📌 Features

### 📊 Basic & Statistical Visualizations

* Bar charts (vertical, horizontal, grouped)
* Box plots
* Pie charts
* Lollipop charts

### 📈 Advanced Visualizations

* Heatmaps
* Violin plots
* Scatter & bubble plots
* Radar (polar) charts

### ⏱ Time Series Analysis

* Moving averages (SMA, EMA)
* RSI indicator
* ARIMA forecasting
* Anomaly detection

### 🌍 Geospatial Visualization

* Cartopy maps
* GeoPandas visualizations

### 🔗 Graph Analysis

* NetworkX graphs
* Shortest path & MST (SciPy)

### 🖼 Image Processing

* Filtering (Gaussian, Sobel)
* Edge detection (Canny)
* Segmentation (Thresholding, K-Means)
* Compression (DCT, FFT, Wavelets)

---

## 🛠 Tech Stack

* Python
* Pandas, NumPy
* Matplotlib, Seaborn
* Scikit-learn
* OpenCV
* NetworkX
* GeoPandas, Cartopy

---

## 📊 Key Insights

* Visualization improves interpretability of complex datasets
* Time-series models reveal trends and anomalies
* Graph analysis uncovers relationships
* Image processing extracts meaningful features

---

## 📌 Use Cases

* Finance (stock analysis)
* Healthcare (patient trends)
